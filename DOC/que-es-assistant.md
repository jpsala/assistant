# Qué Es Assistant

[Inicio](README.md) | [Features](features.md) | [Open Chat](open-chat.md) | [Prompt Picker](prompt-picker.md) | [Prompt Editor](prompt-editor.md) | [Settings](settings.md) | [Ejemplos de uso](ejemplos-de-uso.md)

Assistant es una aplicación de Windows para trabajar sobre texto seleccionado usando prompts y chat contextual.

En vez de copiar texto a mano hacia otra herramienta y volver a pegar el resultado, Assistant se integra al flujo normal de escritura.

Idea principal:

1. seleccionás texto en cualquier app
2. abrís Assistant con un hotkey o desde el tray
3. elegís una acción
4. copiás, pegás o reemplazás el resultado

Qué tipo de tareas resuelve:

- corregir redacción
- resumir texto
- traducir
- reescribir mensajes o mails
- explicar código
- transformar texto con prompts repetibles

Conceptos importantes:

- `selección`: el texto que tenés marcado en otra aplicación
- `prompt`: una instrucción reusable guardada como archivo Markdown
- `Open Chat`: una ventana de chat contextual para iterar
- `Prompt Picker`: una ventana rápida para ejecutar prompts guardados
- `Prompt Editor`: la pantalla para crear y editar prompts
- `Settings`: la pantalla para configurar claves, modelos y hotkeys

Ejemplo simple:

```text
Texto seleccionado:
"Necesito que me confirmes si esto ya está listo porque tenemos que avanzar."

Acción:
Abrir Prompt Picker y ejecutar fix-writing

Resultado esperado:
"Necesito que me confirmes si esto ya está listo, porque tenemos que avanzar."
```
