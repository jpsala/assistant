## 🎯 Issues Críticos (Resumen Rápido)

### 🔴 Issue 2.2: Suffix se Escribe en Pantalla
**Problema:** Al presionar chord muy rápido, suffix se escribe como texto  
**Causa:** `releaseModifiers()` libera teclas antes de registro  
**Solución:** +50ms delay antes de registrar suffix  
**Archivo:** `src/bun/hotkeys.ts:290-335`  
**Documentación:** [Análisis](hotkey-flows-analysis.md#issue-22-escritura-del-suffix-en-pantalla)

---

### 🔴 Issue 3.1: selectAllIfEmpty Falla
**Problema:** Al no tener texto, prompt muestra error  
**Causa:** 40ms insuficiente después de Ctrl+A  
**Solución:** Aumentar a 100ms  
**Archivo:** `src/bun/ffi.ts:350-359`  
**Documentación:** [Análisis](hotkey-flows-analysis.md#issue-31-selectallifempty-no-funciona-consistentemente)

---

### 🔴 Issue 1.1: Race Condition en Captura
**Problema:** Capturas consecutivas se interfieren  
**Causa:** `captureSequence` sin lock/mutex  
**Solución:** Agregar mutex  
**Archivo:** `src/bun/mainview-window.ts:413-427`  
**Documentación:** [Análisis](hotkey-flows-analysis.md#issue-11-race-condition-en-capturasequence)
