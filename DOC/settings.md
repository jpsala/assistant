# Settings

[Inicio](README.md) | [Qué es Assistant](que-es-assistant.md) | [Features](features.md) | [Open Chat](open-chat.md) | [Prompt Picker](prompt-picker.md) | [Prompt Editor](prompt-editor.md) | [Ejemplos de uso](ejemplos-de-uso.md)

`Settings` controla la configuración general de Assistant.

## Qué se puede configurar

- provider por default
- model por default
- API keys
- `Max Tokens`
- hotkeys del sistema
- `Launch on Login`

## Para qué sirve cada cosa

### Provider

Define qué proveedor usar por default cuando un prompt no especifica uno propio.

### Model

Define qué modelo usar por default cuando un prompt no fija uno.

### API Keys

Guardan las credenciales necesarias para hablar con cada proveedor.

### Max Tokens

Controla el tamaño máximo aproximado de respuesta.

### Hotkeys

Definen cómo abrir los flujos principales sin usar el mouse.

Defaults actuales:

- `Prompt Chat`: `Alt+Shift+W`
- `Prompt Picker`: `Alt+Shift+Space`

### Launch on Login

Hace que Assistant se inicie automáticamente cuando el sistema inicia sesión.

## Primer uso recomendado

1. abrí `Settings`
2. cargá tu API key
3. elegí provider
4. elegí model
5. guardá cambios
6. probá `Open Chat` o `Prompt Picker`

## Ejemplo práctico

Configuración simple:

- Provider: `OpenRouter`
- Model: `anthropic/claude-sonnet-4-5`
- Prompt Chat: `Alt+Shift+W`
- Prompt Picker: `Alt+Shift+Space`

Después de eso ya podés usar la app normalmente sobre cualquier texto seleccionado.
