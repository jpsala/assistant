# Prompt Picker

[Inicio](README.md) | [Qué es Assistant](que-es-assistant.md) | [Features](features.md) | [Open Chat](open-chat.md) | [Prompt Editor](prompt-editor.md) | [Settings](settings.md) | [Ejemplos de uso](ejemplos-de-uso.md)

`Prompt Picker` es la forma más rápida de usar Assistant.

Está pensado para cuando ya sabés qué querés hacer y no necesitás abrir un chat.

Cuándo usarlo:

- para corregir texto
- para resumir
- para traducir
- para correr prompts repetitivos

## Cómo funciona

1. seleccionás texto en cualquier app
2. abrís `Prompt Picker`
3. buscás un prompt por nombre
4. lo ejecutás
5. Assistant reemplaza la selección con el resultado

## Qué ventajas tiene

- es más rápido que `Open Chat`
- evita pasos extra
- sirve muy bien para tareas repetidas

## Ejemplos

### Corregir redacción

1. seleccionás un párrafo
2. abrís `Prompt Picker`
3. elegís `fix-writing`

Resultado esperado:

- versión más clara del mismo texto

### Resumir

1. seleccionás un texto largo
2. abrís `Prompt Picker`
3. elegís `summarize`

Resultado esperado:

- resumen corto del contenido

### Traducir al inglés

1. seleccionás texto en español
2. abrís `Prompt Picker`
3. elegís `translate-english`

Resultado esperado:

- versión en inglés lista para usar

## Consejos

- usá nombres de prompt fáciles de recordar
- mantené prompts cortos y claros
- si una tarea requiere mucha ida y vuelta, usá `Open Chat` en vez de `Prompt Picker`
