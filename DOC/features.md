# Features

Assistant hoy tiene estas features principales:

## Open Chat

Sirve para conversar con el modelo usando como contexto el texto seleccionado.

Permite:

- escribir instrucciones libres
- usar prompts guardados con `/nombre-del-prompt`
- copiar la última respuesta
- pegar la última respuesta en la app original
- reemplazar la selección original

## Prompt Picker

Sirve para ejecutar prompts guardados de forma rápida.

Permite:

- buscar prompts por nombre
- navegar con teclado
- ejecutar un prompt sobre la selección actual
- reemplazar texto sin pasar por el chat

## Prompt Editor

Sirve para administrar prompts locales.

Permite:

- crear prompts
- editar prompts
- borrar prompts
- asignar hotkeys
- asignar provider y model por prompt

## Settings

Sirve para configurar el comportamiento general de la app.

Permite:

- guardar API keys
- elegir provider por default
- elegir model por default
- configurar hotkeys
- activar `Launch on Login`

## Tray

La app vive principalmente en el tray de Windows.

Desde el tray se puede:

- abrir `Open Chat`
- abrir `Prompt Picker`
- abrir `Prompt Editor`
- abrir `Settings`
- abrir la guía de uso
- cerrar la app

## Hotkeys

Assistant soporta:

- hotkeys globales del sistema
- hotkeys por prompt
- chords en prompts

Defaults actuales:

- `Open Chat`: `Alt+Shift+W`
- `Prompt Picker`: `Alt+Shift+Space`
