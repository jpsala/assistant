# Open Chat

`Open Chat` es la ventana para trabajar con instrucciones libres y contexto.

Cuándo usarlo:

- cuando querés iterar varias veces
- cuando querés explicar mejor lo que necesitás
- cuando no alcanza con un prompt rápido
- cuando querés generar texto desde cero

Qué toma como contexto:

- si había texto seleccionado, lo usa como contexto inicial
- si no había selección, igual podés usarlo para generar texto nuevo

Qué podés hacer dentro de Open Chat:

- escribir una instrucción libre
- empezar con `/nombre-del-prompt`
- copiar la última respuesta
- usar `Paste in Source App`
- usar `Replace Selected Text`

## Cómo usarlo

### Caso 1: con texto seleccionado

1. seleccioná texto en otra app
2. abrí `Open Chat`
3. escribí una instrucción
4. esperá la respuesta
5. elegí si querés copiar, pegar o reemplazar

Ejemplo:

```text
/fix-writing
Reescribilo para que suene más claro y más breve.
```

### Caso 2: sin texto seleccionado

1. abrí `Open Chat`
2. escribí una instrucción desde cero
3. generá una respuesta
4. usá `Paste in Source App`

Ejemplo:

```text
Escribí un mensaje corto y profesional para pedir una reunión mañana a la tarde.
```

## Botones principales

### Copy Latest

Copia la última respuesta al portapapeles.

### Paste in Source App

Vuelve a la app original y pega la última respuesta en la posición actual del cursor.

### Replace Selected Text

Vuelve a la app original y reemplaza la selección inicial con la última respuesta.

## Ejemplos de uso

### Reescribir un mail

```text
/improve-email
Quiero que suene cordial, firme y corto.
```

### Explicar código

```text
Explicame este código como si estuvieras ayudando a un developer junior.
```

### Traducir

```text
/translate-english
Que suene natural y no literal.
```
